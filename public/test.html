ConsentManager Integration Example

Clean Single-Script Integration

The ConsentManager is integrated with a single script tag in the head of this page:

<script id="consentment-cmp" 
        src="https://app.consentment.com/js/cmp/loader.js" 
        data-settings-id="abcde" 
        data-api-base="https://app.consentment.com/api"
        async></script>

How It Works

The loader script:

  • Automatically blocks scripts requiring consent
  • Loads configuration from the API
  • Loads the appropriate UI component based on configuration
  • Handles consent preferences storage

Testing the Integration

This page has ConsentManager integrated. The consent UI should appear based on your configuration.
